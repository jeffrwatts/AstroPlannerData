Privacy Policy

App: Astro Planner (Island Skies)
Developer: Jeff Watts
Last updated: April 26, 2026

Overview

Astro Planner is a personal astronomy observation planning tool. This policy describes what data the app accesses and how it is used.

Location

The app requests access to your device's GPS location to compute accurate altitude, azimuth, and rise/set times for celestial objects at your observing site. Location data is used only on-device for these calculations and is never transmitted to any server or third party.

Camera

The app declares the camera permission because it includes field-of-view functionality that can reference camera hardware. The camera is not actively used to capture or store images, and no images are transmitted anywhere.

Anthropic API

If you choose to use the AI observing plan feature, your Anthropic API key and the observing parameters you enter (target object, location name, Bortle scale, available time) are sent to the Anthropic API to generate a plan. No other personal data is included in these requests. See Anthropic's privacy policy for details on how they handle API data.

Data Storage

All app data (catalog, equipment configurations, saved plans) is stored locally on your device. No account is required and no data is synced to any cloud service.

Third-Party Services

The app downloads its DSO catalog and astrophotography images from GitHub Pages (hosted by GitHub). No personal data is included in these requests.
